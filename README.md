# BetterTooltip

**BetterTooltip** is a simple, configurable tooltip addon for *Retail WoW* that enhances the default tooltip with useful extra information and cleaner presentation.

## Features

- **Cursor anchoring** for most tooltips
- **Tooltip scaling** to make tooltips easier to read
- **Hide tooltip health bar** for a cleaner look
- **Tooltip coloring** based on your class
- **Show IDs and Icons** for many tooltip types, including:
    - units, spells, mounts, items, toys, currency, quests, macros, achievements, aura tooltips
- **Player tooltip enhancements**, including:
    - mount name, Mythic+ score, target name, guild rank, language
- **LFG / Group Finder language display**
- **Combat-aware tooltip hiding** for selected tooltip types
